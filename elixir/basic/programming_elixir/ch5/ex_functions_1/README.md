# ExFunctions1

Exercises from _Programming Elixir_ by Dave Thomas.

Create and run the functions that do the following:

list_concat.([:a, :b], [:c, :d]) #=> [:a, :b, :c, :d]
sum.(1, 2, 3) #=> 6
pair_tuple_to_list.( { 1234, 5678 } ) #=> [ 1234, 5678 ]
